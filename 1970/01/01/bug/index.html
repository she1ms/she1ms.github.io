未来你这个铸币可能还会再犯的错误

  1. 在写标头时要冒号加空格

    : test

    而非

    :test

  2. 清除缓存

    hexo clean

    而非

    hexo c

  3. 引用图床
    [Files by She1ms (she1ms) - ImageHub - 图仓][def]图像，请使用

    Direct Link: https://s1.imagehub.cc/images/2025/03/04/9db5a5f3cc0810ecaffb96012222792e.jpg

    ​而非
    Viewer Link: https://www.imagehub.cc/image/1b5253d8d7b757870984b0a72f3b63d41607933659.CAKu6q

  4. 添加文章

    请将文章转为.md后直接存入C:\Users\blog\source\_posts

    若原文为.docx，请将文章用格式刷刷一遍，只留标题一二三、正文和引用几类格式，再用Writage转换为.md
